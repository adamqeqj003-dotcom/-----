# 智能书签整理器 (Smart Bookmark Organizer)

🚀 利用 DeepSeek AI 智能整理浏览器收藏夹，自动分类与死链检测。

## ✨ 功能特性

- 📁 **AI 智能分类** - 自动将书签归类到「技术」「新闻」「工具」等文件夹
- 🔗 **死链检测** - 自动识别失效链接并归档到「_死链归档」文件夹
- 🔒 **白名单保护** - 锁定重要书签/文件夹，防止被自动整理
- ⚡ **独立控制台** - 在新标签页运行，不影响正常浏览

## 📦 安装方法

1. 下载本项目（Code → Download ZIP 或 `git clone`）
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `smart-bookmark-organizer` 文件夹

## 🔧 使用方法

1. 点击浏览器工具栏的插件图标
2. 点击「打开控制台」进入独立页面
3. 在设置中填入你的 DeepSeek API Key
4. 点击「测试连接」确认 API 可用
5. 选择整理模式开始整理

## 📝 API Key 获取

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 注册账号并充值
3. 创建 API Key

## ⚠️ 注意事项

- API Key 存储在浏览器本地，不会上传到任何服务器
- 整理过程中请保持控制台标签页打开
- 建议先用「快速整理」测试效果
